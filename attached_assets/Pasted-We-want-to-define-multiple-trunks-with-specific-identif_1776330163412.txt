We want to define multiple trunks with specific identifiers, such as:

First Class Wholesale → “1”
Business Class Wholesale → “2”
Special Bravo → “6”
Special Charlie → “7”

The system should allow selecting a trunk/product first, then pushing rates from the Rate Card into Sippy Soft.

Additionally, we need to implement an email notification system that sends updates to multiple client/customer email IDs. Email configuration should be managed within the Company Profile for different alert types (rate notifications, invoices, disputes, etc.).

A separate Rate Notification Page should also be created to track rate push activity, show client details, system implementation status (App + Sippy Soft), and trigger email notifications upon completion.
