TOP HEADER BAR (Always Visible)

Dashboard   Team Chat

--------------------------------

Why This Is Correct
Dashboard

Dashboard is:

home
landing area
global overview

Operators expect:

one-click access from anywhere

NOT hidden inside a workspace sidebar.

Team Chat

Team Chat is:

cross-functional
always relevant
operational communication

It should behave like:

Slack
Discord
Microsoft Teams

Meaning:

persistent top utility

NOT:

a menu page

LEFT SIDE OF TOP BAR
[Logo] Dashboard
CENTER OF TOP BAR

Workspace navigation:

NOC | Operations | Routing | Intelligence | Analytics

Team Chat
Notifications
Search
Profile
Theme

This is the correct scalable layout.

This separates:

navigation

from:

platform utilities

Which reduces sidebar clutter dramatically.
