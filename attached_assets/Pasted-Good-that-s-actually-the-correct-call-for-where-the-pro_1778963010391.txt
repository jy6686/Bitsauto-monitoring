2. Sidebar becomes “Weighted”, not static

Instead of just ON/OFF:

Each item gets a weight:

{
  label: "AI Ops Center",
  weight: 10,
  contexts: ["noc", "investigation"],
  roles: ["admin", "noc", "management"]
}

Then sidebar renders in order of:

role permission
context relevance
admin priority

So the sidebar becomes:

“What matters right now for this user in this situation”

not:

“what they are allowed to see”
