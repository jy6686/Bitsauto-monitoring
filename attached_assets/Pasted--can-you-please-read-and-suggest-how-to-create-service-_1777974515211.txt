How to Create a Sippy Account via IchiBaan Monitoring Portal
Complete Step-by-Step Team Reference Guide
Overview
The New Sippy Account wizard in the IchiBaan Monitoring portal allows you to create a full SIP customer or vendor account directly on your Sippy Softswitch — without logging into Sippy itself. The wizard has 4 steps and takes roughly 2–3 minutes to complete.

Prerequisite: Admin API credentials must be configured in Settings → Sippy Admin API Credentials before the wizard can create real SIP accounts. If only portal (self-care) credentials are set, the wizard will warn you and create only a portal sub-account — not a true SIP account.

How to Open the Wizard
Log in to the IchiBaan Monitoring portal.
In the left sidebar, click Client / Vendor (under the Clients & Vendors group).
On the page, locate the "New Sippy Account" button (top-right area of the Sippy Accounts tab).
Click it — the 4-step wizard will open as a modal dialog.
Step 1 — Basic Info
This step collects the account identity and credentials.

If no switch is pre-configured in Settings, an inline credentials panel appears:

Field	Required	Description
Sippy URL	Yes	Full URL of your Sippy server, e.g. https://191.101.30.107
Admin Username	Yes	Sippy admin API username (not the portal self-care username)
Admin Password	Yes	Sippy admin API password
Section: Account Identity

Field	Required	Default / Notes

Section: Codec & Media

🔧 1. Controller
export const autoCreateServicePlan = async (req, res) => {
  try {
    const {
      plan_name,
      currency,
      billing_type,
      billing_cycle,
      monthly_fee,
      included_minutes,
      destination_group
    } = req.body;

    // STEP 1: Check if plan exists
    const existingPlan = await sippyGetServicePlan(plan_name);

    if (existingPlan) {
      return res.json({
        success: true,
        service_plan_id: existingPlan.id,
        message: "Service plan already exists"
      });
    }

    // STEP 2: Ensure tariff exists
    let tariff = await sippyGetTariff("AUTO_TARIFF_BASE");

    if (!tariff) {
      tariff = await createTariff({
        name: "AUTO_TARIFF_BASE",
        currency
      });
    }

    // STEP 3: Create Service Plan
    const servicePlan = await createServicePlan({
      name: plan_name,
      tariff_id: tariff.id,
      billing_type,
      billing_cycle,
      monthly_fee,
      included_minutes,
      destination_group
    });

    return res.json({
      success: true,
      service_plan_id: servicePlan.id,
      tariff_id: tariff.id,
      message: "Service Plan created successfully"
    });

  } catch (error) {
    return res.status(500).json({
      success: false,
      error: error.message
    });
  }
};

import axios from "axios";

async function sippyRequest(method, params) {
  const response = await axios.post(process.env.SIPPY_URL, {
    method,
    params
  }, {
    auth: {
      username: process.env.SIPPY_USER,
      password: process.env.SIPPY_PASS
    }
  });

  return response.data.result;
}
