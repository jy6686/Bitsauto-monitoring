# RESTORE FULL COMPANY ONBOARDING WIZARD — IMPLEMENTATION PROMPT

Goal:
Restore the original enterprise-grade telecom onboarding workflow that existed before the navigation/workspace refactor.

Current problem:
The previous 3–6 step onboarding wizard was unintentionally reduced into a lightweight 2-step account flow.

This caused loss of:

* telecom provisioning flow
* reseller onboarding
* white-label setup
* portal/access configuration
* routing initialization
* onboarding validation

The platform needs BOTH:

1. Quick Create Account
2. Full Company Onboarding Wizard

====================================================
ARCHITECTURE DECISION
=====================

KEEP:
Quick Create Account
(Simple 2-step fast flow)

RESTORE:
Full Company Onboarding Wizard
(Multi-step enterprise telecom onboarding)

These are DIFFERENT workflows.
