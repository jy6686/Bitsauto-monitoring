The destination issue and the CDR issue were actually two separate problems:

The Most Important Fix

The live-match fallback using:

gc.caller = destination
gc.callee = CLI

was still using the old mapping.

Fixing it to:

gc.callee = destination (CLD)
gc.caller = ANI (CLI)

was absolutely required.

Without that fix:

manual retries fail,
automatic retries fail,
billing reconciliation fails,
margin calculations fail.
