PHASE 1 — UNIFIED TELECOM RATING ENGINE

services/rating-engine.service.ts

This service becomes the ONLY telecom cost calculation source.

No other module may compute: (duration / 60) * rate

outside this service.

REQUIRED FEATURES
1. Interval-Aware Billing Logic

Support:

1/1 6/6 30/6 60/1 60/6 60/60

Implement:

billedSeconds = initialInterval + ceil( max(0, duration - initialInterval) / incrementInterval ) * incrementInterval

REQUIRED FUNCTION

Implement:

calculateRatedCall()

Input:

{ durationSeconds, buyRate, sellRate, intervalInitial, intervalIncrement, currency, taxPercent, destination, prefix, vendorId, clientId }

Output:

{ durationSeconds, billedSeconds, effectiveCPM, buyCost, sellRevenue, margin, marginPercent, intervalInitial, intervalIncrement, currency, taxAmount, totalCost, rateVersion, timestamp }

EFFECTIVE CPM CALCULATION

Implement:

effectiveCPM = (totalCost / durationSeconds) * 60

This is critical for:

routing intelligence
LCR ranking
profitability
QoS economics
