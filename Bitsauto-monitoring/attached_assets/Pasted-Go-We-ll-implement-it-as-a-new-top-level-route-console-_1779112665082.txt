1. Core Engine (NEW)
server/console/groupAlertsToIncidents.ts

Pure deterministic function:

groupAlertsToIncidents(alerts, verdicts, tickets)
Rules:
Same entity (accountId / vendorId / connectionId)
Time window overlap ≤ 10 minutes
Severity escalation merges upward (info < warning < critical)
Ticket linkage attaches existing open tickets
Verdict overlay attaches latest AI Ops state
Output:
incident {
  id: string
  entity: { accountId, vendorId? }
  severity: "critical" | "warning" | "info"
  title: string
  alerts: Alert[]
  ticketId?: number
  verdict?: string
  startedAt: Date
  lastSeenAt: Date
}
