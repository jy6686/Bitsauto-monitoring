Correct Telecom Billing Formula

The key logic is:

billedSeconds =
  initialInterval +
  ceil(
    max(0, duration - initialInterval)
    / incrementInterval
  ) * incrementInterval

Example:

Duration = 65s
Interval = 60/60

Result:

120 billed seconds

Example:
