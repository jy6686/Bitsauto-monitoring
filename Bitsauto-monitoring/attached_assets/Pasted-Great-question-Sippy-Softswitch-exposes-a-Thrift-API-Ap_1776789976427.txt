Planning Sippy Softswitch file retrieval
There are a few ways to get it, depending on your access level:

Option 1 — It's Already on Your Sippy Server (Easiest)
The Thrift IDL file ships with Sippy Softswitch itself. If you have SSH access to the server, it's usually at one of these paths:

/usr/local/share/sippy/thrift/
/opt/sippy/share/thrift/
/usr/share/sippy/

Ask your server admin to run:

find / -name "*.thrift" 2>/dev/null

and share whatever files come back.

Option 2 — Ask Sippy Support Directly
Email support@sippysoft.com and ask:

"Can you provide the Thrift IDL file for the version of Sippy we're running? We want to build a Thrift API client."

They're generally responsive and this is a legitimate developer request. Mention your licence/account number if you have one.

Option 3 — Check the Sippy Customer Portal / Wiki
If your Sippy subscription includes portal access, log in at your provider's Sippy admin URL and look under:

Developer Docs
API Reference
Downloads
Some resellers publish the Thrift file there.

Option 4 — Extract from a Running Sippy Instance
If you have SSH but can't find the file directly, the Thrift service port is usually 10060 or 19090. You can probe it:

# Check what's listening
ss -tlnp | grep -E '10060|19090'
# Or check the Sippy config for the Thrift port
grep -r thrift /etc/sippy/ 2>/dev/null
