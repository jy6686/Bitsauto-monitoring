WHAT YOU NEED TO IMPLEMENT
STEP 1 — Install Asterisk

On separate VM/server preferred.

Ubuntu:

apt update
apt install asterisk -y
STEP 2 — Create Digit Audio Files

Inside:

/var/lib/asterisk/sounds/en/custom/

Add:

0.wav
1.wav
2.wav
...
9.wav

Optional:

your_otp_is.wav

You can:

record manually
generate via TTS once
reuse forever
STEP 3 — Configure SIP Trunk

Asterisk → Sippy.

Example:

[sippy]
type=peer
host=YOUR_SIPPY_IP
context=from-sippy
disallow=all
allow=ulaw
STEP 4 — Create OTP Dialplan

Example:

[otp-playback]

exten => _X.,1,Answer()
same => n,Playback(custom/your_otp_is)

same => n,Set(OTP=${CALLERID(name)})

same => n,Playback(custom/${OTP:0:1})
same => n,Playback(custom/${OTP:1:1})
same => n,Playback(custom/${OTP:2:1})
same => n,Playback(custom/${OTP:3:1})
same => n,Playback(custom/${OTP:4:1})
same => n,Playback(custom/${OTP:5:1})

same => n,Hangup()
STEP 5 — Originate Call From BitsAuto

BitsAuto sends:

POST /voice-otp

Payload:

{
  "to": "+923001112233",
  "otp": "987432"
}
STEP 6 — BitsAuto Calls Asterisk AMI/API

Example:

Channel: SIP/sippy/+923001112233
CallerID: 987432
Context: otp-playback
Extension: s
Priority: 1

CallerID name carries OTP digits.

STEP 7 — Asterisk Calls Through Sippy

Sippy handles:

LCR
routing
carrier selection
billing
SIP delivery

Asterisk only handles media.

Perfect separation.
